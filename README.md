# 📊 Student Performance Analysis (Day 3)

## 📌 Overview
This project analyzes student performance using Python and Pandas.  
The goal is to apply logic-based operations and extract meaningful insights.

## Tasks Performed
- Explored data using head() and tail()
- Calculated average age
- Identified highest and lowest scoring students
- Found students above average marks
- Added bonus marks to all students
- Added new student data dynamically
- Sorted data based on marks
- Created Pass/Fail status using conditions

## Tools Used
- Python
- Pandas

## Key Insights
- Identified top and lowest performing students
- Counted students above average performance
- Applied logical conditions for grading

## How to Run
pip install pandas
python student_performance_analysis.py
